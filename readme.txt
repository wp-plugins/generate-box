More functionality for the StudioPress Generate Child Theme.

== Description ==
The plugin adds more functionality to the [StudioPress](http://www.studiopress.com/) Generate Child Theme. 	

This plugin is a very handy tool for Generate Child Theme users, it allows you to display the Generate Box on front page, home page, single posts, and everywhere. It also has an option to switch between different color schemes with only a few clicks.

See live example of [Generate Box](http://generatebox.com/).

== Installation ==
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Note: You must have the StudioPress Genesis Framework and/or Generate child theme installed, or simply the plugin won't work!

== Frequently Asked Questions ==

= What is Generate Box plugin? =
Generate Box plugin is inpired by Generate Child Theme for Genesis, it adds more functionality to the Generate Box.

= Is it going to work with my WordPress theme? =
No. You must have the StudioPress Genesis Framework and/or Generate Child Theme installed, or simply the plugin won't work!

Almost every email service that has HTML form.
